Este proyecto es una página web dedicada al **Mundial de Fútbol 2026**, desarrollada como parte del trabajo final de la materia **Desarrollo Web**.  
El sitio presenta información sobre equipos, sedes, calendario y curiosidades del torneo.

## 🧩 Tecnologías utilizadas
- **HTML5** para la estructura del sitio  
- **CSS3 / SASS** para los estilos y diseño responsivo  
- **GitHub Pages** para la publicación del proyecto

## 📁 Estructura del proyecto


## ⚽ Descripción
El sitio web ofrece una experiencia visual atractiva sobre el **Mundial 2026**, incluyendo:
- Información sobre las **sedes y estadios**
- Detalles de los **equipos participantes**
- Calendario de partidos y resultados
- Sección de **redes sociales** para seguir las novedades
